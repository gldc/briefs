# AI/Tech Brief — May 15, 2026

## TL;DR
- OpenAI has integrated Codex capabilities directly into the ChatGPT mobile app, enabling on-the-go task execution and diff viewing.
- Anthropic released Claude Code 2.1.142 with major stability improvements and a new specialized Claude for Legal workflow implementation.
- Researchers disclosed the first public macOS kernel memory corruption exploit targeting Apple's M5 chips.

## Key Stories

**Codex arrives in the ChatGPT mobile app**
OpenAI integrated Codex into the ChatGPT mobile app and updated its developer documentation with setup steps and connected-host behavior.
*Why it matters:* This significantly lowers the barrier to mobile-first development and remote debugging, freeing engineers from desktop IDEs for quick fixes.
[Source](https://developers.openai.com/codex/changelog)

**Claude Code 2.1.142 released**
Anthropic released an update to Claude Code introducing new `claude agents` flags for dispatched background sessions and defaulting Fast mode to Opus 4.7.
*Why it matters:* The robust background session support and shift to Opus 4.7 streamlines multi-agent development and boosts performance for high-speed automated coding.
[Source](https://code.claude.com/docs/en/changelog)

**Claude for Legal**
Anthropic published a new guide and open-source implementation for using Claude in specialized legal workflows.
*Why it matters:* This highlights a major push to move frontier LLMs from general reasoning into highly specialized, compliance-heavy professional industries.
[Source](https://github.com/anthropics/claude-for-legal)

**First public macOS kernel memory corruption exploit on Apple M5**
Security researchers successfully published the first kernel memory corruption exploit for the M5 chip architecture.
*Why it matters:* This marks a critical milestone in hardware security research for Apple's newest silicon, demonstrating that the M5's memory protections can be bypassed in specific kernel contexts.
[Source](https://blog.calif.io/p/first-public-kernel-memory-corruption)

**Access to frontier AI will soon be limited**
A new analysis argues that access to cutting-edge AI models will increasingly be limited by economic realities and national security policies rather than purely technical hurdles.
*Why it matters:* It points to a looming shift from open AI proliferation to a bifurcated market where geopolitical and financial barriers restrict access to the most powerful models.
[Source](https://writing.antonleicht.me/p/cut-off)
